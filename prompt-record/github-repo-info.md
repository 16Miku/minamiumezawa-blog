## 6. 项目亮点 (用于 README badge 或介绍段落)

- 📚 **完整归档**：174 篇博客 + 819 张图片，时间跨度 2018.02 – 2026.05
- 🔒 **离线可用**：所有图片本地化，零远程依赖，原站下线亦可访问
- ✅ **可验证准确**：文本逐字符 + 图片字节级 MD5 三层校验，与原版完全一致
- 🎨 **视觉还原**：乃木坂紫主题，响应式设计，Lighthouse 全项 100
- 🛡️ **工程韧性**：17 个 pytest 测试，幂等构建，一键复现
- 🚀 **自动部署**：push 即触发 Render 重新部署
